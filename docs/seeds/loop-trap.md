# 为什么换了那么多人，同样的问题还在？​‌​​‌‌​‌

换人没用。换策略没用。换赛道没用。每年都在"重新出发"，每年都回到原点。

这不是人的问题。是**结构锁死**。

## 什么是结构锁死

一个系统有三个要素：目标、路径、边界。

当边界把路径全框死了，目标就跟你没有关系了。不管换多少人、换多少策略——人走在路径上，路径被边界框死，人只能原地画圈。

你看到的是"换了人还是不行"。你感受到的是"为什么我总是回到这里"。

原因：**你没碰边界。你一直在边界里面调人和策略。**

## 为什么你看不到边界

边界最狡猾的地方——它是隐形的。它不是"不能做"，是"做不到"。但"做不到"和"不能做"长得一模一样。

你试了100种方法，99种失败了。你以为第100种会成功。但如果边界是隐形的，第100种跟第1种在同一个边界里面。失败不是方法不对，是边界没动。

## 怎么破

不是换人。不是换策略。

是**换坐标系**。

你站在自己的坐标系里，边界是隐形的。换一个不站在你坐标系里的人来看——他看不到你的目标和路径，但他能看到你的边界。因为他不站在里面。

这个人不能是你的团队、你的朋友、你的同行。他们都跟你共享同一个坐标系。必须是**外面的人**。
