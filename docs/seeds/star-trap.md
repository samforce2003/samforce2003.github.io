# 为什么越成功越像被困住了？​‍‌‌​‌‌‌

一个人爬到足够高的位置之后，最大的困境不是敌人变强了，是**没有下一个台阶了**。

## 三种监狱

有一种监狱没有铁栏杆。你站在最高处，往下看是来路，往上看是天花板。你动不了——不是因为你不够强，是因为这个位置的结构决定了**你不能再往上**。

这叫**峰值监狱**。

还有一种监狱更隐蔽。你身边的人越来越多，但所有人都在仰视你。仰视你的人没办法给你关系——他们只能给你服务。服务不是关系。你得到的是"被需要"，不是"被看见"。你在人群中，但没有一个人是平视。

这叫**孤岛监狱**。

第三种最狠。外面所有人都在说"你应该怎样"——你应该突破、你应该转型、你应该放下身段、你应该保持格调。每个人给你一个方向，每个方向都指向他们自己的利益。你的路被别人的建议堵死了。

这叫**多向监狱**。

## 为什么常规手段解决不了

换团队、换策略、换人设——全部是在峰值监狱里重新装修。墙没动。

找更多人帮你——是在孤岛监狱里增加仰视你的人。更孤独。

听更多建议——是在多向监狱里增加新的方向箭头。更晕。

## 唯一的出口

是一个**不仰视你的人**。

不是来帮你解决问题的人。帮你解决问题的人，解决问题后会走——因为他就是来解决问题的。你要的是一个人，他能看到你的结构——不是你的资源、不是你的位置、不是你的人设——是你被困住的**结构**。然后告诉你："你在这里，门在那边。"

这种人不能追。追来的都在仰视。只能等。你要做的是让这个人知道：我在这里，门开着。
